Calcolatore Valore Assoluto

Calcola il valore assoluto di qualsiasi numero

Puoi inserire numeri negativi e decimali

Visualizzazione Grafica

Il valore assoluto di un numero è la sua distanza dall'origine sulla retta numerica, indipendentemente dalla direzione.

Matematicamente: |x| = x se x ≥ 0, e |x| = -x se x < 0

Esempi

-5
|−5| = 5
3.14
|3.14| = 3.14
-2.5
|−2.5| = 2.5
0
|0| = 0
