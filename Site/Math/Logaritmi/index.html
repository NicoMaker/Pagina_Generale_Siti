Cos'è un Logaritmo?

Il logaritmo di un numero in una data base è l'esponente a cui la base deve essere elevata per ottenere quel numero.

Formula: logb(x) = y significa by = x

Esempi Comuni

  • log10(100) = 2 perché 102 = 100
  • log2(8) = 3 perché 23 = 8
  • loge(x) = ln(x) (logaritmo naturale)
